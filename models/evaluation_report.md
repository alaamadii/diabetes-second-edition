# Model Evaluation Report

## Overview
This report compares Logistic Regression, Random Forest, and SVM using a unified ML Pipeline.

## Performance Metrics

- **Logistic Regression**: Accuracy = 0.7143 (AUC = 0.8230)
- **Random Forest**: Accuracy = 0.7597 (AUC = 0.8147)
- **SVM**: Accuracy = 0.7532 (AUC = 0.7924)

## Conclusion

The best performing model is **Logistic Regression** with an AUC of 0.8230 and Accuracy of 0.7143.
It provides the best balance of True Positive Rate and False Positive Rate, as seen in the ROC curve and overall accuracy.
